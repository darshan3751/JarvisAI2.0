### 🖐️ Jarvis AI 2.0 — Gesture Highlights

> My latest project lets you **control your PC with hand gestures** — completely hands-free.

| Gesture | Action |
|---------|--------|
| 👌 Snap fingers | Toggle gesture mode ON/OFF |
| ✋ Palm UP | Scroll up |
| 🤚 Palm DOWN | Scroll down |
| ☝️ Index clockwise | Volume up |
| ☝️ Index anticlockwise | Volume down |
| ✌️ Two fingers | Play / Pause |
